= RailsAdmin HTML5 Multiple File Upload Custom Action

Helper gem, used to add a custom action to RailsAdmin for multiple file uploading to a photo gallery. More info in the related blog post: {Multiple file upload with HTML5 and RailsAdmin}[http://blog.powpark.com/2013/10/16/multiple-file-upload-with-html5-and-rails-admin/]
